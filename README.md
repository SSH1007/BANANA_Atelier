* 프로젝트명: 바나나공방

    - 작품 큐레이션
    - 자품 경매
* 주요 기술
    - WebRTC
    - JWT Authentication
    - REST API
* 프로젝트 진행 기간 : 2023.01.03일(화) ~ 2023.02.17(금)
